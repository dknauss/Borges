# Coordinate first-wave official language packs

## Context

The repository contains current seed PO/MO files for 19 locales, but official
public language availability is determined by generated WordPress.org language
packs. As of the 2026-06-14 audit, the live plugin page reported one translated
locale and showed English (US) plus Russian.

## First wave

Prioritize reviewed official packs for:

-   `fr_FR`
-   `de_DE`
-   `es_ES`
-   `pt_BR`
-   `ja`

## Acceptance Criteria

-   A qualified human reviewer or locale-team path is identified for each first
    wave locale before strings are submitted for approval.
-   Translations are submitted through translate.wordpress.org, locale-team PTE,
    or CLPTE workflows.
-   No unreviewed machine translations are submitted.
-   Docs continue to point to the live WordPress.org Languages list as the
    canonical official-availability source.
-   README/readme wording is updated only after a generated language pack appears
    publicly.
